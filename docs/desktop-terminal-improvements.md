## 核心变化

### 1. 从「Setup Center」到「桌面终端」

不再只是配置安装工具，而是集 **AI 聊天 + 配置管理 + 状态监控 + 技能市场 + IM 通道** 于一体的完整桌面客户端。

### 2. 中英文双语 / 国际化（i18n）

- 基于 `react-i18next` 实现全面国际化
- 自动检测系统语言，中文系统默认中文，其他默认英文
- 顶栏一键切换语言按钮
- 所有界面文字、配置标签、状态信息完整翻译

### 3. 中国本土化

- **国内 LLM 服务商一等公民**：阿里云 DashScope（通义千问）、月之暗面 Kimi、MiniMax、DeepSeek、硅基流动
- **国内 PyPI 镜像**：内置清华 TUNA、阿里云源，配置页面一键切换
- **中文 IM 通道**：飞书、企业微信、钉钉、QQ 原生支持
- **中文人格角色**：贾维斯、虚拟女友/男友、家人、管家等角色均有完善的中文人格定义

### 4. 真人化 & 角色系统

OpenAkita 最具特色的功能之一——**Agent 不只是工具，更像一个有性格的真人助理**：

- **8 种预设角色**：默认助手、商务顾问、技术专家、私人管家、虚拟女友、虚拟男友、家人、贾维斯
- **三层人格架构**：预设基础 → 用户偏好学习 → 上下文自适应
- **活人感模式**：
  - 主动问候（根据时段和用户习惯）
  - 任务跟进提醒
  - 记忆回忆（"上次你提到想学吉他..."）
  - 频率智能控制，不打扰但有存在感
  - 安静时段自动静音
- **偏好自动挖掘**：每轮对话 LLM 自动分析，提取用户性格偏好，每日归纳更新身份文件
- **越用越懂你**：持续积累用户画像，个性化沟通风格

### 5. 桌面聊天助手

全功能 AI 聊天界面，对标 Cursor/ChatGPT 体验：

- 流式输出 + Markdown 渲染
- 多模态输入（文本 / 图片 / 语音 / 文件）
- LLM 端点选择器（支持指定模型，不自动回退）
- Thinking 内容展示
- Plan 模式
- 斜杠命令
- Cursor 风格输入框

### 6. 简易流程化配置

配置流程 9 步引导，每一步精简聚焦：

1. **开始** — 平台信息概览 + 步骤说明
2. **工作区** — 创建配置隔离空间
3. **Python** — 一键安装内置 Python
4. **安装** — 一键 venv + pip 安装
5. **LLM 端点** — 弹窗式添加：选服务商 → 输入 Key → 拉取模型 → 选择 → 确定
6. **IM 通道** — 按需启用 Telegram/飞书/企业微信/钉钉/QQ
7. **工具与技能** — Skills 管理 + MCP/桌面自动化开关
8. **Agent 与系统** — 角色选择 + 活人感 + 核心参数
9. **完成** — 检查清单 + 一键启动

### 7. 状态监控面板

紧凑一屏展示：
- 服务运行状态 + 工作区信息 + 自启动开关
- LLM 端点健康状态表格（Key/连通性/延迟）
- IM 通道状态
- 技能统计
- 服务日志实时查看

### 8. UI/UX 改善

- 侧栏可折叠（只留图标），适应小屏
- 去紫色，改为专业蓝色主题
- 弹窗 backdrop blur + slide up 动画
- 端点表格化展示（替代卡片式）
- 所有 emoji 替换为 SVG 图标
